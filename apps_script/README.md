# Google Apps Script版 楽天注文LINE通知

この版はPCを起動しっぱなしにしなくても動きます。Google Apps Scriptが5分おきにGmailを確認し、新しい楽天注文メールがあればLINEに通知します。

## できること

- Gmailの注文メールを自動確認
- `order@rakuten.co.jp` の `【楽天市場】注文内容ご確認（自動配信メール）` だけを対象にする
- `[商品]` 欄から商品名、SKU、サイズ、価格だけを抜き出す
- LINE公式アカウントの友だちへ通知する
- 一度通知したメールは重複通知しない

## 1. Apps Scriptを作る

1. [Google Apps Script](https://script.google.com/) を開く
2. 「新しいプロジェクト」を作成
3. 最初からある `コード.gs` の中身を全部消す
4. このプロジェクトの `apps_script/Code.gs` の中身を全部コピーして貼り付ける
5. プロジェクト名を `楽天注文LINE通知` などに変更して保存

## 2. LINEトークンを入れる

Apps Scriptに貼り付けたコードの中で、次の部分を探します。

```javascript
function setLineToken() {
  const token = 'ここにチャネルアクセストークンを貼り付け';
```

`ここにチャネルアクセストークンを貼り付け` の部分を、LINEのチャネルアクセストークンに置き換えます。

例:

```javascript
function setLineToken() {
  const token = '長いチャネルアクセストークン';
```

保存したら、上部の関数選択で `setLineToken` を選び、実行します。

実行後、トークンはApps Scriptの保存領域に保存されます。保存できたら、コード上のトークン文字列は再び `ここにチャネルアクセストークンを貼り付け` に戻して保存しても大丈夫です。

## 3. 抽出テスト

上部の関数選択で `testParser` を選び、実行します。

ログに次のような内容が出ればOKです。

```text

日時: 2026-06-12 08:55:46
```

## 4. Gmailから実際に探してLINE通知する

上部の関数選択で `notifyRakutenOrders` を選び、実行します。

初回はGoogleの権限確認が出ます。許可してください。

必要な権限:

- Gmailのメール検索と本文読み取り
- 外部サービスへの接続、つまりLINE APIへの送信
- Apps Script内の保存領域の使用

成功すると、LINEに注文商品通知が届きます。

## 5. 自動実行を設定する

上部の関数選択で `setupTrigger` を選び、実行します。

これで5分おきに `notifyRakutenOrders` が自動実行されます。PCを閉じていても動きます。

## 送信先について

このコードはLINE Messaging APIの `broadcast` を使います。つまり、そのLINE公式アカウントを友だち追加している全員に送ります。

今のように友だちが自分1人だけなら問題ありません。将来ほかの人を友だち追加する場合は、ユーザーID指定の個別送信方式へ切り替えてください。

## 重複通知について

通知済みのGmailメッセージIDをApps Scriptの保存領域に記録します。同じ注文メールは再通知しません。

過去の記録を消して最初からやり直したい場合は、Apps Scriptの「プロジェクトの設定」からスクリプトプロパティを開き、`PROCESSED_MESSAGE_IDS` を削除してください。

## 自動実行を止める

上部の関数選択で `deleteTriggers` を選び、実行します。

これで5分おきの自動実行が止まります。
